## Variations

### Default (`ul`, `ol`, `.list`)

0.25rem of margin on the bottom of list items

```html
<ul>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ul>
```
<div className="panel">
  <ul>
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
  </ul>
</div>

### Loose

0.5rem of margin on the bottom of list items

```html
<ul class="list list--loose">
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ul>
```
<div className="panel">
  <ul className="list list--loose">
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
  </ul>
</div>

### Tight

No margin at the bottom of list items

```html
<ul class="list list--tight">
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ul>
```
<div className="panel">
  <ul className="list list--tight">
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
  </ul>
</div>
