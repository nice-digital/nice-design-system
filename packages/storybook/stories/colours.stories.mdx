# Colours

Colours are split into two sets:

- Core colours (\_settings-colours-nice.scss)
- Semantic colours (\_settings-colours-semantic.scss)

The core colours should consist of NICE's colour palette.
The semantic colours take these colours and apply them to specific scenarios, e.g. link colour, border colours etc.
