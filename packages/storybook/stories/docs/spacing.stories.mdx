# Spacing

Settings can be found within NDS Core at [settings/settings-spacing.scss](https://github.com/nice-digital/nice-design-system/blob/main/components/nds-core/scss/settings/_settings-spacing.scss).

There are a number of set sizes to be used for margins/padding/borders, listed below:

| Var | Pixels | Rems | Sample code | Width |
| --- | ------ | ---- | ----------- | ----- |
| $xx-small | 2px | 0.125rem | `core.rem(core.$xx-small)` | <Block width="xx-small" /> |
| $x-small | 4px | 0.25rem | `core.rem(core.$x-small)` | <Block width="x-small" /> |
| $small | 8px | 0.5rem | `core.rem(core.$small)` | <Block width="small" /> |
| $medium | 16px | 1rem | `core.rem(core.$medium)` | <Block width="medium" /> |
| $large | 32px | 2rem | `core.rem(core.$large)` | <Block width="large" /> |
| $x-large | 48px | 3rem | `core.rem(core.$x-large)` | <Block width="x-large" /> |
| $xx-large | 64px | 4rem | `core.rem(core.$xx-large)` | <Block width="xx-large" /> |
