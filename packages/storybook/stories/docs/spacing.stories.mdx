# Spacing

Settings can be found within NDS Core at [settings/settings-spacing.scss](https://github.com/nice-digital/nice-design-system/blob/main/components/nds-core/scss/settings/_settings-spacing.scss).

There are a number of set sizes to be used for margins/padding/borders, listed below:

| Var | Size | Px equivalent | Sample code |
| --- | ---- | ------------- | ----------- |
| $nds-spacing-xx-small | 2 | 1px | core.rem(core.$nds-spacing-xx-small) |
| $nds-spacing-x-small | 4 | 3px | core.rem(core.$nds-spacing-x-small) |
| $nds-spacing-small | 8 | 7px | core.rem(core.$nds-spacing-small) |
| $nds-spacing-medium | 16 | 15px | core.rem(core.$nds-spacing-medium) |
| $nds-spacing-large | 32 | 30px | core.rem(core.$nds-spacing-large) |
| $nds-spacing-x-large | 48 | 45px | core.rem(core.$nds-spacing-x-large) |
| $nds-spacing-xx-large | 64 | 60px | core.rem(core.$nds-spacing-xx-large) |
