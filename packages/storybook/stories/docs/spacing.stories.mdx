# Spacing

Settings can be found within NDS Core at [settings/settings-spacing.scss](https://github.com/nice-digital/nice-design-system/blob/main/components/nds-core/scss/settings/_settings-spacing.scss).

There are a number of set sizes to be used for margins/padding/borders, listed below:

| Var | Pixels | Rems | Sample code | Width |
| --- | ------ | ---- | ----------- | ----- |
| $nds-spacing-xx-small | 2px | 0.125rem | `core.rem(core.$nds-spacing-xx-small)` | <Block width="xx-small" /> |
| $nds-spacing-x-small | 4px | 0.25rem | `core.rem(core.$nds-spacing-x-small)` | <Block width="x-small" /> |
| $nds-spacing-small | 8px | 0.5rem | `core.rem(core.$nds-spacing-small)` | <Block width="small" /> |
| $nds-spacing-medium | 16px | 1rem | `core.rem(core.$nds-spacing-medium)` | <Block width="medium" /> |
| $nds-spacing-large | 32px | 2rem | `core.rem(core.$nds-spacing-large)` | <Block width="large" /> |
| $nds-spacing-x-large | 48px | 3rem | `core.rem(core.$nds-spacing-x-large)` | <Block width="x-large" /> |
| $nds-spacing-xx-large | 64px | 4rem | `core.rem(core.$nds-spacing-xx-large)` | <Block width="xx-large" /> |
