## Paragraphs

### Standard paragraphs

<Example>
	<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nullam interdum nibh pellentesque, blandit nunc vitae, semper ipsum.</p>
	<p>In libero nisi, laoreet vel lacinia vel, malesuada id eros. Pellentesque tempor interdum tortor non malesuada. Curabitur nec consequat ex, rhoncus dictum sapien. In varius nisl metus. Curabitur sollicitudin venenatis elit, a feugiat nulla. Vivamus luctus in dui ut maximus. Morbi quis nisl est. Ut a ex lectus. Etiam interdum malesuada felis, vitae malesuada felis sagittis a.</p>
</Example>

##### Usage:

- Use a `p` element, or
- Apply a class of `p`, or
- Use the `nds-p` SASS mixin

- Apply a class of `lead` to a `p` element, or
- Use the `nds-lead` SASS mixin

## Links

### Default link style

<Example>
	<p><a href="https://www.nice.org.uk">Here is a link to NICE!</a></p>
</Example>

### Inverse link style

##### Usage:

- Default style for all links (i.e. `a` elements)
- use the `nds-link-default` mixin

<Example>
	<p style={{background: "#333", padding: "10px"}}><a className="link--inverse" href="https://www.nice.org.uk">Here is an inverted link to NICE!</a></p>
</Example>

##### Usage:

- use the `link--inverse` class
- use the `typography.link-inverse` mixin
